## Troubleshooting

### Common Issues

**Calendar won't load:**
- Check GitHub Pages is enabled
- Wait 5-10 minutes after enabling
- Clear browser cache

**Photos won't upload:**
- Check file size (<5MB)
- Use JPG, PNG, or GIF
- Try one photo at a time

**Language won't switch:**
- Click toggle button in top-right
- Refresh page
- Check localStorage enabled

**Data disappeared:**
- Restore from backup export
- Check not in incognito mode
- Verify same browser
